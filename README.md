# Jumanji-MERN-web-application
umanji est une application web développée avec la stack MERN et TensorFlow.js . Elle permet aux utilisateurs d’explorer les espèces animales menacées en Tunisie. Grâce à un chatbot intelligent, les utilisateurs peuvent envoyer une image d’un animal afin d’analyser s’il s’agit d’une espèce en danger ou non
